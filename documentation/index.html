Markup

How To Edit Color

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

h1 {
   color: #someColor;
}

Change Structure

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

header h1 {
   color: #someColor;
}

How to add new sections to homepage

If you want to add new pages to homepage, you should use the following structure.

<section class="main"> 
<article class="column">.....</article> 
<article class="column">.....</article> 
<article class="column">.....</article> 
<article class="column">.....</article> 
</section> 

<section class="main"> 
<article class="column">.....</article> 
............
<!-- Use max 4 articles in one section --> 
</section>

How To Change the Contact Forms Recipient E-mail

If you want to change the Recipient E-mail, open up the processForm.php file and change the following code: 

define( "RECIPIENT_NAME", "YOUR NAME" ); 
define( "RECIPIENT_EMAIL", "YOUR MAIL" );

How To Change Background Images

If you want to change the homepage background images, open up the custom.js file and change the following code: 

/* Article Backgrounds */
function background() {
    "use strict";
    if ($(window).width() > 480) {
        $("#home").backstretch("images/concert.jpg");
        $(".about").backstretch("images/4.jpg");
        $(".albums").backstretch("images/2.jpg");
        $(".news").backstretch("images/1.jpg");
        $(".events").backstretch("images/3.jpg");
        $(".blog").backstretch("images/1.jpg");
        $(".contact").backstretch("images/3.jpg");
    }
}
